Task 5: Data Cleaning with Pandas
 Overview

This project demonstrates basic data reading and cleaning using Python (Pandas) on the Titanic dataset.
It shows how Python can efficiently replace manual Excel-based data cleaning.

Tools

Python

Pandas, NumPy

Google Colab / Jupyter Notebook

 Steps Performed

Loaded dataset using pd.read_csv()

Analyzed structure with .head() and .info()

Handled missing values (median & mode)

Removed duplicates

Converted data types

Created new features (Age_Group, Fare_Band)

Exported cleaned dataset to CSV

 Files

Task5_Cleaning.ipynb

Titanic_Cleaned.csv

 Outcome

Gained hands-on experience in data preprocessing using Pandas, preparing datasets for analysis and modeling.
